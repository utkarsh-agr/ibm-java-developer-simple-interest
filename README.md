# Simple Interest Calculator

## Project Description

This project provides a simple interest calculator implemented as a Bash shell script. The calculator computes simple interest based on user-provided inputs including principal amount, rate of interest, and time period.

## Features

- Interactive command-line interface
- Calculates simple interest using the formula: Simple Interest = (Principal × Rate × Time) / 100
- Easy to use and understand
- Lightweight Bash script requiring no additional dependencies

## Usage

Run the script and follow the prompts to enter:
- Principal amount
- Rate of interest (annual percentage)
- Time period (in years)

The script will calculate and display the simple interest.

## Project Information

**Project Name:** Simple Interest Calculator  
**Language:** Bash  
**Purpose:** Educational project for learning Git and GitHub workflows

## Getting Started

1. Clone this repository
2. Navigate to the project directory
3. Make the script executable: `chmod +x simple-interest.sh`
4. Run the script: `./simple-interest.sh`
